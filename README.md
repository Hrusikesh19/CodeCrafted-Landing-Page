# CodeCrafted

A modern, responsive landing page built using HTML and CSS.

## Overview

CodeCrafted is a static frontend project focused on clean layout structure, responsive design, and professional UI composition. The goal of this project was to practice building a structured SaaS-style landing page using core web technologies without any frameworks.

## Tech Stack

- HTML5  
- CSS3  
- Flexbox  
- Responsive Design  

## Features

- Responsive layout  
- Modern navigation bar  
- Hero section with call-to-action  
- Clean and structured UI  
- Mobile-friendly design
